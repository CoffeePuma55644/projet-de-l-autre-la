Apprendre le CSS : Rendre le Web Joli !

CSS, ça veut dire Cascading Style Sheets. C'est le langage qui s'occupe de l'apparence de tes pages HTML. Si HTML est le squelette, CSS est l'habillage : les couleurs, les polices, la mise en page... tout ce qui rend un site agréable à regarder !

Sans CSS, le web ressemblerait à des documents textes un peu tristes. Avec CSS, on peut faire des merveilles ! (Bon, au début, on fait des trucs simples, mais c'est déjà super !)

Comment ça marche ?

Le CSS fonctionne avec des "règles". Une règle CSS dit : "Prends cet élément HTML (par exemple, tous les titres <h1>) et applique-lui ces styles (par exemple, mets-les en bleu et en grande taille)".

On écrit généralement le code CSS dans un fichier séparé (comme notre style.css !) et on le relie à nos pages HTML. C'est pratique, parce qu'on peut changer l'apparence de tout un site en modifiant juste un fichier !

Un petit exemple de CSS

Imaginons qu'on veuille que tous nos paragraphes (<p>) soient en couleur vert clair et nos titres <h2> en orange. Dans notre fichier CSS, on pourrait écrire : 

/* Ceci est un commentaire en CSS */

p {
    color: lightgreen; /* Texte des paragraphes en vert clair */
    font-size: 16px;   /* Taille de la police */
}

h2 {
    color: orange;     /* Texte des titres h2 en orange */
    text-align: center;/* Centrer les titres */
}

Et voilà ! Tous les <p> et <h2> de notre site (qui utilise ce CSS) prendraient ces styles. Les couleurs qu'on utilise sur CE site sont définies dans style.css, tu peux y jeter un œil !

Le CSS, c'est un peu comme jouer avec des Lego : on assemble des propriétés pour créer le look qu'on veut. Il y a plein de propriétés à découvrir : background-color pour la couleur de fond, font-family pour la police, margin et padding pour les espaces...
